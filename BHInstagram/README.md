# BHInstagram
An awesome tweak for Instagram!

# Features
- Hide Ads
- No suggested post
- Show Like count
- Confirm like
- Copy description
- Download Videos
- Save profile image
- Keep deleted message
- Remove last seen
- Remove screenshot alert
- Unlimited replay of once story
- Disable Story Seen Receipt
- Padlock
